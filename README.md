# A Game for ~~Every~~ Some Consoles

## About

Project aimed at learning more about low-level development, processor architecture and game design throughout history.

I try to use the technology used at the time, since some of them have little to no content available, i used what is most common in the respective homebrew community.

The idea is to create at least one game for each [console generation](https://en.wikipedia.org/wiki/History_of_video_game_consoles).
